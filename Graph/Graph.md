# 1. 인접 행렬

연걸된 노드를 2차원 배열로 표현하는 것이다. 방향성이 없다면 각 노드의 인덱스에 1로, 방향성이 있다면 연결된 노드에 1을 넣는다.

## 단점

인접행렬은 메모리르 과도하게 낭비한다는 점이다. 또한 순차적으로 각 노드에 접근한다면 불필요한 연산을 많이하게 된다.

- 방향 O

    ```c++
    graph[i][j] = 1;
    graph[j][i] = 1;
    ```

- 방향 X

    ```c++
    graph[i][j] = 1;
    ```
